# CUT04 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT04.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 0:30〜0:45（15秒）／BPM 95.7（1拍 ≈ 0.627秒、15秒で約24拍）
- 出演: MIU、AYA、NANA
- 場所: 渋谷、スクランブル交差点から路地へ向かう通り、夜
- 感情アーク: 目的地のない自由さ → 好奇心の芽生え → 路地へ折れる決断
- 演出方針: 「背後ワイド→横からのトラッキング→路地入口へのパン」を4パネルに分解

## パネル構成一覧

| Panel | Timecode | 拍(目安) | ショットサイズ | カメラ | 構図の要点 |
|---|---|---|---|---|---|

## Panel 1 — 背後トラッキング

- ショットサイズ／レンズ: WS
- カメラワーク: 3人の背後を一定速度で追走
- 構図:
```
┌────────────────────────────┐
│                              │
│   ●MIU  ●AYA  ●NANA(背中)    │
└────────────────────────────┘
```
- 芝居: 3人が並んで歩く
- 補足: 大通りのネオンを背景に残す

## Panel 2 — 横からの並走

- ショットサイズ／レンズ: MS
- カメラワーク: 側方からの並走トラッキング
- 構図:
```
┌────────────────────────────┐
│  ●MIU    ●AYA    ●NANA      │
│      （笑い合いながら歩く）      │
└────────────────────────────┘
```
- 芝居: 軽くじゃれ合いながら会話
- 補足: 大通りから路地への光量変化をこのパネルの後半から開始

## Panel 3 — NANA、路地を指す

- ショットサイズ／レンズ: MCU、NANA中心
- カメラワーク: NANAへ軽く寄る
- 構図:
```
┌────────────────────────────┐
│                              │
│      ●NANA(指差す)           │
│      視線: 路地の方向          │
└────────────────────────────┘
```
- 芝居: NANA「こっち行ってみない？」
- 補足: 指差す方向が次パネルのパン先と一致させる

## Panel 4 — 路地へ折れる

- ショットサイズ／レンズ: WS
- カメラワーク: NANAの指差し方向へパン、3人が追従して曲がる
- 構図:
```
┌────────────────────────────┐
│  大通り→          →路地      │
│      （3人が路地へ折れていく）   │
└────────────────────────────┘
```
- 芝居: 3人が路地へ入っていく
- 補足: 光量がここで明確に落ち着いたトーンへ切り替わる

## パネル間の繋ぎ（編集意図）

- Panel1→2: 同一トラッキングの中でカメラ位置を背後→横へ緩やかに移動(ワンカットでも可)
- Panel4→CUT05: 路地へ折れる動作の勢いのままCUT05(ダイナー発見)へカット

## 生成上のメモ

- Panel1-2の3人歩行は歩調の同期(BPMに合わせた足取り)が重要。生成時に不自然な歩幅差が出ないよう留意
