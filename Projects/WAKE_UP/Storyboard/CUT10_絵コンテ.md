# CUT10 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT10.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 2:20〜2:40（20秒）／BPM 95.7（1拍 ≈ 0.627秒、20秒で約32拍）
- 出演: MIU、AYA、NANA（+ 人混みの中にKAIに似た人物のシルエット）
- 場所: 渋谷、帰り道の雑踏
- 感情アーク: 日常会話への復帰 → 既視感の閃き → 確信を持てない揺らぎ
- 演出方針: 「歩く3人→人混み奥のシルエットへフォーカス送り→MIUが振り返るが誰もいない」を4パネルに分解

## パネル構成一覧

| Panel | Timecode | 拍(目安) | ショットサイズ | カメラ | 構図の要点 |
|---|---|---|---|---|---|

| 2 | 2:26–2:31 | 9.6〜17.6拍 | WS→フォーカス送り | 背景の人混み | 遠くのシルエット |

## Panel 1 — 歩く3人

- ショットサイズ／レンズ: MS
- カメラワーク: 横からの並走トラッキング
- 構図:
```
┌────────────────────────────┐
│   ●MIU   ●AYA   ●NANA       │
│      （会話しながら歩く）        │
└────────────────────────────┘
```
- 芝居: 3人が歩きながら会話を続ける(セリフ内容は限定しない)
- 補足: なし

## Panel 2 — 遠くのシルエット

- ショットサイズ／レンズ: WSから背景へラック(フォーカス送り)
- カメラワーク: 固定、フォーカスのみ背景へ移動
- 構図:
```
┌────────────────────────────┐
│  ●MIU ●AYA ●NANA(手前ボケ)   │
│         ▓（逆光のシルエット）   │
└────────────────────────────┘
```
- 芝居: 背景にKAIに似た逆光のシルエットが一瞬映る
- 補足: 顔の判別を意図的に避ける(既知の演出意図)

## Panel 3 — MIUの反応

- ショットサイズ／レンズ: MCU(MIU)
- カメラワーク: 固定
- 構図:
```
┌────────────────────────────┐
│      ●MIU（はっと振り返る）     │
└────────────────────────────┘
```
- 芝居: MIUだけが気づいて振り返る(セリフなし、息をのむ気配)
- 補足: AYA・NANAはこの変化に気づかない

## Panel 4 — 消えたシルエット

- ショットサイズ／レンズ: WS
- カメラワーク: MIUの視線方向へのパン
- 構図:
```
┌────────────────────────────┐
│  →   →    (誰もいない)  →    │
└────────────────────────────┘
```
- 芝居: MIUが探すが人混みに紛れて見えない
- 補足: 「まさか」という確信の持てない既視感を残す

## パネル間の繋ぎ（編集意図）

- Panel2→3: シルエットの一瞬とMIUの反応はテンポよく短く繋ぐ
- Panel4→CUT11: MIUの振り返りの間からCUT11(HINA登場)へ
