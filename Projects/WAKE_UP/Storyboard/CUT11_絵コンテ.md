# CUT11 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT11.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 2:40〜3:00（20秒）／BPM 95.7（1拍 ≈ 0.627秒、20秒で約32拍）
- 出演: MIU、AYA、NANA、HINA
- 場所: 渋谷、帰り道の歩道
- 感情アーク: 何気ない帰り道 → すれ違いの気配 → 言葉にならない既視感
- 演出方針: 「すれ違うHINAの全身→MIUの振り返り→HINAの後ろ姿が遠ざかる」を4パネルに分解

## Panel 1 — すれ違う瞬間

- ショットサイズ／レンズ: MS
- カメラワーク: 横からの並走、HINAと3人がフレーム内で交差
- 構図:
```
┌────────────────────────────┐
│  ●MIU ●AYA ●NANA →  ←●HINA │
└────────────────────────────┘
```
- 芝居: HINAが白い服で3人とすれ違う。AYA・NANAは気にせず歩き続ける
- 補足: HINAの白い服だけが周囲の暖色の中でやや浮いて見える

## Panel 2 — MIUだけ振り返る

- ショットサイズ／レンズ: MCU(MIU)
- カメラワーク: 固定
- 構図:
```
┌────────────────────────────┐
│      ●MIU（足を止め振り返る）   │
└────────────────────────────┘
```
- 芝居: MIUだけが足を止めて振り返る
- 補足: AYA・NANAは前を歩き続けている(画面手前でフレームアウト)

## Panel 3 — MIUのセリフ

- ショットサイズ／レンズ: CU(MIU)
- カメラワーク: 静止
- 構図:
```
┌────────────────────────────┐
│      ●MIU（探すような表情）     │
└────────────────────────────┘
```
- 芝居: MIU「（小さく、独り言のように）……どこかで、会った気がする」
- 補足: 静かで小さな、けれど心に残る違和感

## Panel 4 — HINAの後ろ姿

- ショットサイズ／レンズ: WS
- カメラワーク: 固定〜わずかな引き
- 構図:
```
┌────────────────────────────┐
│   ○HINA（後ろ姿、遠ざかる）     │
│      （振り返らず歩き去る）      │
└────────────────────────────┘
```
- 芝居: HINAは振り返らずそのまま歩き去り、雑踏に紛れていく
- 補足: なし

## パネル間の繋ぎ（編集意図）

- Panel1→2: すれ違いの一瞬とMIUの反応は非対称の演出(MIUだけが反応)を明確に見せる
- Panel4→CUT12: MIUのセリフの余韻でCUT12(エンディング)へ
