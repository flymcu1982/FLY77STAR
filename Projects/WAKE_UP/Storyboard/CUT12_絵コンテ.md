# CUT12 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT12.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 3:00〜3:19（19秒、楽曲末尾まで）／BPM 95.7（1拍 ≈ 0.627秒、19秒で約30拍）
- 出演: MIU、AYA、NANA
- 場所: 渋谷、帰り道
- 感情アーク: 解決しないまま続く日常 → 一瞬の振り返り → 次への静かな予感
- 演出方針: 「3人の後ろ姿→MIUの最後の振り返り→街の夜景へ引いていく」を4パネルに分解

## パネル構成一覧

| Panel | Timecode | 拍(目安) | ショットサイズ | カメラ | 構図の要点 |
|---|---|---|---|---|---|
| 1 | 3:00–3:06 | 0〜9.6拍 | MS | 後ろ姿 | 3人が歩き続ける |
| 2 | 3:06–3:10 | 9.6〜16拍 | MCU(MIU) | 振り返り | 一度だけ |
| 3 | 3:10–3:13 | 16〜20.8拍 | MS | 前を向く | 歩調を合わせる |
| 4 | 3:13–3:19 | 20.8〜30拍 | WS(クレーンアップ) | 引き | 夜景全体へ |

## Panel 1 — 後ろ姿

- ショットサイズ／レンズ: MS
- カメラワーク: 背後からのトラッキング
- 構図:
```
┌────────────────────────────┐
│   ●MIU  ●AYA  ●NANA(背中)    │
└────────────────────────────┘
```
- 芝居: 3人が並んで歩き続ける
- 補足: なし

## Panel 2 — MIUの振り返り

- ショットサイズ／レンズ: MCU(MIU)
- カメラワーク: 固定
- 構図:
```
┌────────────────────────────┐
│      ●MIU（一度だけ振り返る）   │
└────────────────────────────┘
```
- 芝居: MIUが一度だけ後ろを振り返り、小さく息をつく
- 補足: この一瞬にCUT10・CUT11の既視感の余韻を重ねる

## Panel 3 — 前を向く

- ショットサイズ／レンズ: MS
- カメラワーク: 背後トラッキングへ復帰
- 構図:
```
┌────────────────────────────┐
│   ●MIU  ●AYA  ●NANA         │
│      （歩調を合わせ直す）        │
└────────────────────────────┘
```
- 芝居: MIUが前を向き、AYA・NANAと歩調を合わせる
- 補足: なし

## Panel 4 — クレーンアップ

- ショットサイズ／レンズ: WS→ELS、クレーンアップ
- カメラワーク: 上昇しながら後退。3人が画面内で徐々に小さくなる
- 構図:
```
┌────────────────────────────┐
│   ░渋谷の夜景全体░            │
│        ●●●(小さくなる3人)     │
└────────────────────────────┘
```
- 芝居: なし
- 補足: 楽曲アウトロ(BPM95.7)に合わせてフェードアウト。エンドカードで「物語は次回へ続く」を示す静止テロップを想定(Palmier側で追加)

## パネル間の繋ぎ（編集意図）

- Panel1→2→3: 振り返りは切らずにワンカット内の芝居の推移として繋ぐ
- Panel4: 本作の最終パネル。テロップ挿入スペースを画面上部または下部に確保しておく

## 生成上のメモ

- 俯瞰の夜景生成は情報量が多く、Palmier側でのテロップ配置スペースを確保する必要がある(既知の注意点、CUT12.md 4者レビュー参照)
- クレーンアップの尺は楽曲アウトロと合わせる微調整が必要
