## Panel 1 — 眠る渋谷、青い時刻（0:00–0:05.02、1×8）

```
+--------------------------------------+
|   ~~~~ 深い藍色の空 ~~~~             |
|______________________________________|
|  ::: 街の灯 :: ネオン残光 :::        |
|  ██ ▓▓ ██ ▓▓▓ ██ ▓ ███ ▓▓ ██        |
|  ██ ▓▓ ██ ▓▓▓ ██ ▓ ███ ▓▓ ██        |
|      → 車のライトが一筋 →           |
+--------------------------------------+
        ▼ カメラはゆっくり前進降下
```

- **ショット**: 超ロング（空撮俯瞰）
- **カメラ**: 等速の前進降下。揺れなし
- **光**: 空＝深い藍。街＝点在するネオンと街灯の暖色
- **芝居**: なし。街そのものの「眠り」を見せる
- **補足**: エピローグ（SHIBUYA_RETURN）で回収する「原点の画」。実在ランドマークが写り込まない架空の渋谷風スカイライン

## Panel 2 — 降下、路地へ（0:05.02–0:10.04、2×8）

```
+--------------------------------------+
| ▓▓|      ビルの壁      |▓▓          |
| ▓▓|  ___________       |▓▓          |
| ▓▓| | ネオン残光 |      |▓▓          |
|   |  ‾‾‾‾‾‾‾‾‾‾‾       |            |
|     ≈≈ 濡れた路面の反射 ≈≈          |
+--------------------------------------+
        ▼→ ビル間を縫って路地の高さへ
```

- **ショット**: ロング→ミディアムロング（高度が下がり続ける）
- **カメラ**: 単一の滑らかなパスで降下＋前進
- **光**: ピンク／シアンのネオン残光が濡れたアスファルトに反射。人気はほぼ無し
- **芝居**: なし。シャッターの降りた店、換気口の湯気などのディテール
- **補足**: 看板類はすべて判読不能な色面として処理（ノーロゴ徹底）

## Panel 3 — クラブ入口（0:10.04–0:15.06、3×8）

```
+--------------------------------------+
|      ▒▒ 判読不能なネオン形状 ▒▒     |
|   ______________________________     |
|  |  ██████  ドア  ██████  |          |
|  |  ██████ (黒/重厚) ██▌ ←隙間から  |
|  |________________________|  暖色光  |
|     青い路面   ⇡ もや               |
+--------------------------------------+
        →→ 地上高さの前進ドリー
```

- **ショット**: ミディアム。ドアが画面中央へ
- **カメラ**: 極めて安定した前進ドリー（地上目線）
- **光**: 手前＝青い時刻の冷たい光／ドアの隙間＝漏れる暖色＋薄いヘイズ。この寒暖対比が本作の色彩テーマの初出
- **芝居**: ドア越しに低音の気配（音響側で処理）
- **補足**: `LOC_CLUB_EXT01`（仮）。店名・外観の正式デザインは Director 判断待ち

## Panel 4 — 3人、夜から出てくる（0:15.06–0:20.08、4×8）★重要Panel

```
+--------------------------------------+
|  ▒▒ ネオン ▒▒   ／暖色の逆光＼      |
|   _______＼｜開いたドア｜／______    |
|      ○AYA    ○MIU    ○NANA         |
|     （左）  （中央）  （右）         |
|      笑いの余韻、自然な歩み          |
|            MIU→ 視線は画面外右       |
+--------------------------------------+
     カメラ: 停止→わずかにプルバック
```

- **ショット**: ミディアム3ショット（顔がしっかり判別できるサイズを死守）
- **カメラ**: 前進を止め、3人が出てくるのに合わせてわずかに引く（迎え入れる動き）
- **光**: 背後＝ドアからの暖色逆光（髪のエッジライト）／正面＝青い時刻の環境光。両者が顔の上で混ざる
- **芝居**: ドアが外開きに開く（強拍同期）→ AYA・MIU・NANA が笑いながら自然に出てくる → 3人がゆるく横並びになり歩みが緩む → 最終拍で MIU だけ画面外右（ダイナー方向）へ視線
- **並び**: 左から AYA／MIU（センター）／NANA。CANON のグループ内役割（MIU＝リードボーカル/センター）を立ち位置で反映
- **補足**: 重要Panel該当理由 — **主人公初登場**（SE77NTH. 3人の本編初登場）。Director Notes: `WAKE_UP_CANON_SCENE01_PANEL04_クラブ出口3人_DirectorNotes.md`
- **Scene 2 への接続**: MIU の視線が Scene 2 Panel 3「始発まであそこで待とう！」の第一声を受ける視線バトン
