# CUT02 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT02.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 0:08〜0:18（10秒）／BPM 95.7（1拍 ≈ 0.627秒、10秒で約16拍）
- 出演: MIU、AYA
- 場所: 渋谷スクランブル交差点付近、夜（CUT01と同一エリア）
- 感情アーク: 一人の静止(CUT01の続き) → 気づき → 合流の高揚感
- 演出方針: 「人混み奥からAYAが手前へ抜けてくるトラッキング→MIUが振り返る→二人が向き合う」を4パネルに分解

## パネル構成一覧

| Panel | Timecode | 拍(目安) | ショットサイズ | カメラ | 構図の要点 |
|---|---|---|---|---|---|
| 1 | 0:08–0:11 | 0〜4.8拍 | WS | 人混み奥からのトラッキング | AYAが群衆を縫って手前へ |
| 2 | 0:11–0:13 | 4.8〜8拍 | MCU | 反応ショット | MIUが振り返る |
| 3 | 0:13–0:16 | 8〜12.8拍 | MS | 並走〜収束 | AYAが駆け寄り距離が詰まる |
| 4 | 0:16–0:18 | 12.8〜16拍 | MS(二人) | 静止 | 合流、笑顔、セリフ |

## Panel 1 — AYA、人混みから

- ショットサイズ／レンズ: WS、標準(35mm感)
- カメラワーク: 奥から手前へのトラッキング。AYAが群衆をすり抜けてくる
- 構図:
```
┌────────────────────────────┐
│   →   →    ●AYA  →   →     │
│      →         →           │
│  （画面奥から手前へ移動）       │
└────────────────────────────┘
```
- 芝居: AYAが小走りで人混みを縫って進む
- 補足: CUT01最終パネルのMIUの視線先とマッチカットになるよう、AYAの進行方向をCUT01の画面外方向と一致させる

## Panel 2 — MIU、振り返る

- ショットサイズ／レンズ: MCU
- カメラワーク: ほぼ静止、微細なパンでMIUの振り返り動作を追う
- 構図:
```
┌────────────────────────────┐
│                              │
│        MIU（振り返る）        │
│        視線: 画面奥へ          │
└────────────────────────────┘
```
- 芝居: MIUが気配に気づき振り返る
- 補足: CUT01のラストと直結する反応ショット

## Panel 3 — 合流までの距離

- ショットサイズ／レンズ: MS
- カメラワーク: 並走気味のトラッキングから軽い収束
- 構図:
```
┌────────────────────────────┐
│  ●AYA(手を上げる)             │
│              ●MIU(待つ)      │
└────────────────────────────┘
```
- 芝居: AYAが手を軽く上げて合図、MIUも表情が緩む
- 補足: 距離が詰まる速度をBPMのノリに合わせる

## Panel 4 — 二人、合流

- ショットサイズ／レンズ: MS(二人)
- カメラワーク: 静止
- 構図:
```
┌────────────────────────────┐
│   ●MIU        ●AYA          │
│      （向き合って小さく笑う）    │
└────────────────────────────┘
```
- 芝居: 顔を見合わせて小さく笑う。AYA「(小声で)待った？」
- 補足: セリフのタイミングはパネル開始直後

## パネル間の繋ぎ（編集意図）

- Panel1→2: AYAの接近とMIUの振り返りは同時進行のクロスカット、またはワンカット内のパン割りでも可
- Panel3→4: 距離が詰まる動きは切らずに連続させる
- 最終パネル→CUT03: AYAのセリフの余韻でCUT03(3人集合、NANA登場)へ直接カット

## 生成上のメモ

- Panel1(トラッキング)は背景の人流生成が鍵。Panel4(二人)はSoul IDの一貫性を特に重視(MIU・AYA双方)
