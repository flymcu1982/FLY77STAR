# CUT03 絵コンテ（映画レベル） — WAKE UP（SE77NTH.）

Story Bible確定を受けて作成。生成プロンプト本体は `Storyboard/CUT03.md` を参照。Studio OS標準制作ワークフロー Version 1 Step 3.5(詳細: GitHub `AI_WORKFLOW_V1.md`)で作成。

## 基準情報

- 尺: 0:18〜0:30（12秒）／BPM 95.7（1拍 ≈ 0.627秒、12秒で約19拍）
- 出演: MIU、AYA、NANA
- 場所: 渋谷スクランブル交差点付近、夜
- 感情アーク: 二人の合流(CUT02の続き) → 3人目の登場 → チームが揃う高揚感
- 演出方針: 「NANAの登場→3人が向き合う→横並びで歩き出す前の3ショット」を4パネルに分解

## Panel 1 — NANA、駆け込む

- ショットサイズ／レンズ: MS
- カメラワーク: 固定〜わずかなパンでNANAの侵入を追う
- 構図:
```
┌────────────────────────────┐
│         ●NANA(駆け込む)      │
│   ●MIU      ●AYA            │
└────────────────────────────┘
```
- 芝居: NANAが小走りで合流。NANA「お待たせー！」
- 補足: セリフはこのパネルの終盤

## Panel 2 — ハイタッチ

- ショットサイズ／レンズ: MCU、手元寄りの切り返し
- カメラワーク: 固定、3人の手が触れる瞬間を中心に
- 構図:
```
┌────────────────────────────┐
│    ✋(NANA)   ✋(MIU/AYA)     │
└────────────────────────────┘
```
- 芝居: 軽いハイタッチ、笑い声
- 補足: 手の動きのタイミングをBPMに軽く同期させる

## Panel 3 — 3人の笑顔

- ショットサイズ／レンズ: CU、3人分の切り返し
- カメラワーク: 固定の切り返しショット3連続
- 構図:
```
┌────────────────────────────┐
│                              │
│       (顔のクローズアップ)     │
│                              │
└────────────────────────────┘
```
- 芝居: 3人それぞれの笑顔を短く見せる
- 補足: NANAのほくろ(左目下)の位置をSoul ID基準として維持

## Panel 4 — 3ショットへ引く

- ショットサイズ／レンズ: WS
- カメラワーク: ゆっくり引く(プルバック)
- 構図:
```
┌────────────────────────────┐
│   ●MIU   ●AYA   ●NANA       │
│    （横並びで静止）             │
└────────────────────────────┘
```
- 芝居: 3人が横並びになり、次の「歩き出す」への構えができる
- 補足: CUT04冒頭の背後トラッキングへ繋がる並び順(左からMIU・AYA・NANA)を固定する

## パネル間の繋ぎ（編集意図）

- Panel1→2→3: テンポよく短いカットで畳みかける(BPMのノリを画面に反映)
- Panel4→CUT04: 3人が横並びになった瞬間でCUT04(歩き出す)へカット
