## 2. カット調整仕様（接続点1〜8）

TC=タイムコード（目安）。F=フレーム。実映像優先。

### 接続1: 渋谷上空 → 路地上空
- 完全な黒画面まで落とさない。黒へ深く落ちる部分があれば短縮
- **4〜6F の短いクロスディゾルブ**、または非常に弱い方向ブラー
- 長いフェードアウト/インは**禁止**。「同じ街の中を進む」感覚を維持

### 接続2: 路地上空 → クラブ外観
- カメラの前進/降下が**続いている途中**で切る
- 路地映像の静止した末尾を削る／クラブ外観の動き出し前の静止も削る
- 基本ストレートカット。必要時のみ **2〜4F ディゾルブ**

### 接続3: クラブ外観 → 3人が出てくる
- 現在のDirector編集の重ね方を基準
- クラブ外観を長く見せすぎない。ドアが開く流れが自然な位置で接続
- 重なりは短く。**人物が二重に見える長いディゾルブ禁止**
- **この接続は「音の変化」を主役に**（映像トランジション最小）

### 接続4: クラブ前ワイド → NANA会話ショット
- 黒フェード**不使用**。基本ストレートカット。必要なら **2〜3F ディゾルブ**
- **Jカット**: NANA「今日……」の音声を切替の **3〜5F 前**から先行させ、切断感を隠す

### 接続5: NANA会話ショット → 街歩き
- **長いディゾルブ禁止**（人物二重回避）
- 台詞終了後、自然な反応を **0.5〜0.8秒**残す
- 歩く肩・髪・足の動きが近い瞬間で切る。基本ストレート、必要でも **2〜4F 以内**
- 会話ショットの止まり切った末尾は不使用／街歩きの動き出し前静止も削る
- 目安トリム: 前素材末尾 **6〜10F**、次素材冒頭 **3〜6F**（動き優先で調整）

### 接続6: 街歩き → ダイナーに気づく
- 視線/歩行方向がつながる位置で切る。長いディゾルブ不使用。基本ストレート

### 接続7: ダイナーを見る → ダイナー外観
- 順番変更禁止（人物の視線 → 視線の先の外観）
- 基本ストレート。外観は **1.5〜2秒**目安、建物を長く見せない。最も見やすい部分のみ

### 接続8: ダイナー外観 → 近づく → ドアを開ける
- 同方向へ進む位置で接続。AYAが手を伸ばす**動作途中**で次へ切る案を検討
- 動作が止まってから切らない。黒フェード**禁止**

## 3. クラブ環境音プラン（既存ライブラリのみ・壁越し低音中心）

**方針**: クラブ外観→3人退出を、映像でなく音で自然につなぐ。明確な楽曲は不使用、壁越しの低いキック＋ベース中心。可能ならローパス処理。

| 位置 | 処理 | 音量イメージ |
|---|---|---|
| 路地上空 終盤 | 遠くから低いクラブ音を導入 | 非常に小さく |
| クラブ外観 | 壁越し「ズン、ズン」を継続、高音/メロディは抑制、ローパス | 小 |
| ドア開放の瞬間 | クラブ音を **+4〜6dB** だけ自然に上げる（急激にしない）。適切なドア音があればごく小さく | 一瞬だけ増 |
| 3人退出後 | **1〜1.5秒**かけて遠ざける。台詞前に十分下げる | 減衰→小 |

**検索候補（既存ライブラリ）**: muffled club music / nightclub ambience / bass through wall / distant club bass / sub bass pulse / club door ambience

## 4. 街・雨の環境音プラン（一本の音の橋）

- 渋谷上空〜ダイナー入口まで薄い街＋雨の環境音を**継続**（カットごとに完全に切らない）
- 音量と距離感のみ場面で変化。街歩きは雨・足音・遠い車中心
- 環境音は**台詞より大きくしない**
- ダイナー外観の **6〜8F 前**から、既存素材に店内小環境音がある場合のみ非常に薄く先行可。**新規店内音生成は禁止**

**検索候補**: rainy city ambience / night street ambience / distant traffic

## 5. 台詞処理（NANA「今日、楽しかったね」）

- 台詞の変更・生成・差し替え**禁止**
- 頭と末尾に短いフェード
- 映像より **3〜5F 先行**するJカットを試す
- 台詞中はクラブ低音を下げる。台詞後に別の声/不要音声を追加しない
- 台詞終了後は雨と街の音だけに自然に戻す

## 6. トランジション制限

- **許可**: ストレートカット / Jカット / Lカット / 2〜6F の短いディゾルブ
- **禁止**: 黒画面への長いフェード / 人物が長時間二重のディゾルブ / 強いフラッシュ / グリッチ / 大きなズーム / 過剰発光 / 同一トランジションの反復
- 原則: トランジションを増やさず、**カット位置と音の継続**で自然につなぐ
