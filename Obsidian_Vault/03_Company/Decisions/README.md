# Decisions
（正式な意思決定ログ）

「いつ・何を・なぜ決めたか」を1件1ファイルで残します。Founder's Journal(個人の想い)やDiscovery Log(気づき)とは違い、会社として確定した決定のみを置きます。

## ファイル名

`YYYY-MM-DD_決定内容.md`

## 書き方

[[99_Templates/意思決定ログ|テンプレート]] を使ってください。

## 例外: 未決事項リスト

Vaultレビューなどで「YUの判断待ちの項目」がまとまって出てきた場合は、`YYYY-MM-DD_未決事項リスト.md` として1ファイルにまとめてよい(frontmatterに `状態: 未決` を明記)。個々の項目が決まったら、その項目には✅を付け、必要なら通常の意思決定ログとして別ファイルに切り出す。確定していない項目をこのフォルダに置いてよい、唯一の例外がこのパターン。
