# Obsidian_Vault について

このフォルダは FLY77STAR STUDIO の Obsidian Vault(社内Wiki用のノート集)です。

Obsidianアプリでこのフォルダを「Vaultとして開く」と、リンクを辿って閲覧できます。

最初に開くファイル: `00_Start_Here/ここから読む.md`

## 位置づけの注意

このVaultは会社の「記憶」を保管する場所であり、正式なルールブックではありません。

制作ルールの正本(せいほん)は常にリポジトリ直下の `OPERATING_MANUAL.md` 等です。Vault内の文章とGitHubのルールファイルが矛盾する場合は、GitHub側が優先されます。

(詳細: `AI_RULES.md` の Obsidian の項目)
