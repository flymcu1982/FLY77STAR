# ここから読む

このVaultを初めて開いた人(人間・AI社員どちらも)向けの入り口です。上から順に読めば、5分でFLY77STARの全体像がつかめます。

## 毎日ここを見る

社長(YU)は朝と夜、まず [[09_Daily_Studio_Report/_Index|Daily Studio Report]] （毎日AM/PMに社長が確認する30秒日報）を開いてください。今日の状況を30秒で把握できます。
