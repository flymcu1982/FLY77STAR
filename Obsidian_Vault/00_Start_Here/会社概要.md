# 会社概要

## Vision / Mission

AIと人の創造力を組み合わせ、映画品質の映像作品を効率的かつ継続的に制作する。

(出典: GitHubリポジトリ `OPERATING_MANUAL.md`)

## Core Values

- 感情を最優先する。
- 技術は作品のために使う。
- 妥協ではなく改善を積み重ねる。
- 一つの作品ごとに制作フローを進化させる。
- AIは代役ではなくチームメンバー。

## 制作パイプライン

企画 → 脚本 → 絵コンテ → キャラクター制作 → 映像生成 → 仮編集 → 本編集 → カラー → 音響 → 書き出し → 広報

## Studio OS の5レイヤー

1. 制作 （企画・脚本・絵コンテ・キャラクター・CUT設計を管理する層）
2. 編集 （Palmierへ渡す編集指示・尺・同期・フェード・音響を管理する層）
3. 広報 （YouTube/TikTok/Instagram向けの書き出しと投稿素材を管理する層）
4. 品質管理 （映像・音・余韻・ルール準拠をチェックする層）
5. 自動化 （フォルダ生成・構成検証・編集指示書生成・GitHub Actionsを整備する層）

## この会社の記録の仕方(文化)

FLY77STARは「何を作ったか」だけでなく「なぜそう考えたか」「何を発見したか」も資産として残します。

- Founder's Journal （創業者の想い・考えを残すノート）
- Discovery Log （今日の気づきノート）

の2つが、その文化を支える仕組みです。詳しくは [[Vault運用ルール]] を参照。
