# 組織図

FLY77STARは人間1名+AI社員(ツール)で構成されています。全員「AI社員」として、担当領域内では自律的に判断・提案します。最終判断は常にYUです。

| 名前 | 役割 | 一言でいうと |
|---|---|---|
| YU | CEO | 最終判断者。すべての意思決定はYUの承認で確定する |
| ChatGPT / Claude(チャット) | 企画・戦略・世界観・脚本 | 物語と方針を考える担当 |
| Claude Code | CTO | GitHub・フォルダ・自動化を管理する担当(このVaultを整備しているのもこの役割) |
| Higgsfield / Nano Banana | キャラクター・ビジュアル生成 | キャラクターの絵を作る担当 |
| Seedance / Gemini Omni Flash | MV・動画生成 | 映像とキャラクターの演技を作る担当 |
| Palmier | 動画編集 | 素材をつないで作品に仕上げる担当 |
| ElevenLabs | ナレーション・ボイス制作 | セリフや声を作る担当 |
| Obsidian | 会社の記憶・社内Wiki | このVault自体。判断の理由や過去の記録を残す担当 |
| GitHub | 資産保管庫 | コード・ドキュメント・軽量素材を正式に保管する場所 |
| Netlify | 公式サイト公開 | 完成したものを世に出す担当 |

役割の詳細・得意なこと・禁止事項は GitHub上の `AI_RULES.md`(正本)を参照してください。
