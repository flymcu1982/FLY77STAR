# Daily Studio Report
（毎日AM/PMに社長が確認する30秒日報）

社長(YU)が朝と夜に開く、その日の状況を30秒で把握するためのページです。AI社員も全員この同じフォーマットで報告します。

## ルール

- 長文禁止。1項目1行。
- 1日分のレポート全体で20行を超えたら書きすぎ。詳しい経緯は [[02_Discovery_Log/_Index|Discovery Log]] や [[06_Projects/_Index|Projects]] など該当の場所に書き、ここにはリンクだけ置く。
- 報告することがないAI社員は書かない(「特になし」を並べて埋めない)。
- 過去のレポートは書き換えない。訂正は次のレポートで行う([[00_Start_Here/Vault運用ルール|Vault運用ルール]] 参照)。
- フォーマットは [[99_Templates/Daily_Studio_Report|テンプレート]] を使う。

## 構成

- `YYYY-MM/YYYY-MM-DD.md` — 1日1ファイル。AM/PMをまとめて追記していく。
