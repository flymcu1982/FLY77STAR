# Distances QC Checklist

## Before Palmier Export

- [ ] CUT01〜CUT18が順番通りに配置されている。
- [ ] CUT06、CUT12、CUT17のセリフ同期を確認した。
- [ ] CUT17の余韻を切っていない。
- [ ] CUT18の肩を寄せるあと約1秒を残している。
- [ ] セリフ後に急なカットがない。
- [ ] BGMがセリフを邪魔していない。
- [ ] 環境音が残っている。
- [ ] 9:16の構図で顔や重要な手元が切れていない。
- [ ] Kodak Vision3 / Blue Hour / Warm Skin / Natural Contrastの方針に合っている。

## Before Final Delivery

- [ ] SNS用タイトル案を作成した。
- [ ] サムネ候補を作成した。
- [ ] 完成動画を `Exports/` に保存した。
- [ ] 作品別の書き出しを `Projects/Distances/Export/` に保存した。
