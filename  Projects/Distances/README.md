# Distances

FLYSTAR77 STUDIO作品 `Distances` の制作フォルダ。

## Read First

1. `../../OPERATING_MANUAL.md`
2. `../../PRODUCTION_BIBLE.md`
3. `PROJECT_BRIEF.md`
4. `CUT_LIST.json`
5. `Edit/EDIT_PLAN.md`

## Standard Folders

- `CUT/`: CUT01〜CUT18の動画素材
- `Import/`: 未整理の素材投入フォルダ
- `Unsorted/`: CUT番号が判断できない素材の確認待ち
- `Watch/`: 監視スクリプト用の素材投入フォルダ
- `Audio/`: BGM、環境音、効果音
- `Dialogue/`: セリフ、ナレーション
- `Reference/`: 参考資料
- `Prompt/`: Dreamina用プロンプト
- `Edit/`: Palmier編集指示
- `Storyboard/`: 絵コンテ
- `Color/`: カラー方針
- `QC/`: 品質管理
- `Delivery/`: SNS投稿素材
- `Export/`: 作品別書き出し
