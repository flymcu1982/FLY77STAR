# FLYSTAR77 Folder Rules

この `RULE.md` をFLYSTAR77制作管理における最優先ルールとして扱う。

## Standard Folders

- `Projects/`: 作品を保存する。
- `Assets/`: 共通素材を保存する。
- `Templates/`: 共通プロンプトを保存する。
- `Exports/`: 完成動画を保存する。
- `ProductionBible/`: 制作ルールを保存する。

## Project Folder Rule

作品ごとの制作データは `Projects/作品名/` に保存する。

例:

```text
FLYSTAR77/
└── Projects/
    └── Distances/
        ├── Import/
        ├── Unsorted/
        ├── CUT/
        ├── Audio/
        ├── Dialogue/
        ├── Reference/
        ├── Prompt/
        ├── Edit/
        ├── Storyboard/
        ├── Color/
        ├── QC/
        ├── Delivery/
        ├── Export/
        └── project.json
```

不足しているフォルダがある場合は、自動で提案する。
