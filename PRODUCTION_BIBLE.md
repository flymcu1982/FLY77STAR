# FLYSTAR77 Production Bible

このファイルは `OPERATING_MANUAL.md` とあわせて、FLYSTAR77 STUDIOの最優先ルールとして扱う。

## Rule Priority

1. `OPERATING_MANUAL.md`
2. `PRODUCTION_BIBLE.md`
3. `ProductionBible/RULE.md`

`ProductionBible/RULE.md` は詳細な制作ルール置き場として扱う。

## Core Rules

- `Projects/`: 作品を保存する。
- `Assets/`: 共通素材を保存する。
- `Templates/`: 共通プロンプトを保存する。
- `ProductionBible/`: 制作ルールを保存する。
- `Exports/`: 完成動画を保存する。
- `Archive/`: 完了・保留・旧版の制作物を保存する。
- 不足しているフォルダやファイルがあれば提案する。
