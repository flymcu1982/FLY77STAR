# FLYSTAR77 Palmier Rules

Palmierは仮編集とタイムライン生成を担当する。

## Palmier Inputs

Palmierへ渡す基本ファイル:

- `Projects/作品名/CUT/`
- `Projects/作品名/Audio/`
- `Projects/作品名/Dialogue/`
- `Projects/作品名/Edit/EDIT_PLAN.md`
- `Projects/作品名/project.json`

## Editing Rules

- CUTを順番に読み込む。
- セリフ同期ポイントを優先する。
- BGMはセリフを邪魔しない。
- 環境音は可能な限り残す。
- フェードは余韻を壊さない範囲で使う。
- セリフ後は急にカットしない。

## Export Rules

- TikTok / Instagram Reels: 9:16
- YouTube: 4K
- 完成動画は `Exports/` に保存する。
- 作品別の中間書き出しは `Projects/作品名/Export/` に保存する。
