## Overview

This document defines the SNS implementation and operation rules for FLY77STAR. across YouTube, TikTok, Instagram, and X.

**Core Principle:** Keep the brand unified across platforms, but treat YouTube differently from TikTok and Instagram.

- YouTube: do not create a new official channel; renew the existing `@yuyama6664` channel as the FLY77STAR. official channel.
- TikTok / Instagram: new official accounts may be created.
- X: manage with a separate handle policy because `@FLY77STAR_STUDIO` exceeds the 15-character limit.
- Official Gmail: may be newly created for platform operations.

## Existing YouTube Video Policy

- Existing YouTube videos must not be deleted.
- Every existing video must be reviewed and categorized into one of the following four groups.

### 1. Public

- Keep the video public when it still supports current brand presentation.

### 2. Unlisted

- Keep the video accessible by direct link, but remove it from the main public channel surface.

### 3. Private

- Keep the video archived inside the channel without public visibility.

### 4. Re-edit / Reuse

- Reuse parts of the video through re-editing for Shorts, teasers, archive projects, or retrospective content.

**Important Rules**

- Deletion is prohibited.
- Any visibility change between Public, Unlisted, and Private requires `YU最終承認`.
- Any reuse plan that materially changes public presentation also requires `YU最終承認`.

## YouTube Migration Workflow

1. Create the official Gmail account candidate.
2. Verify ownership, login access, and recovery settings for `@yuyama6664`.
3. Check handle availability in the order below:
   - `@FLY77STAR_STUDIO`
   - `@FLY77STAR_OFFICIAL`

5. Classify each video as Public, Unlisted, Private, or Re-edit / Reuse.
6. Prepare the channel renewal package:
   - new channel name
   - new icon
   - new banner
   - new description
   - playlist plan
   - video classification sheet

8. Apply the approved production changes to the existing channel.
