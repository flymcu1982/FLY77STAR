# FLYSTAR77 Prompt Guide

Dreaminaなどの映像生成AIへ渡すプロンプトは、感情を中心に設計する。

## Prompt Sections

全CUT共通で以下を含める。

- Cinematic Style
- Character
- Location
- Camera
- Scene
- Acting
- Emotion
- Director's Notes
- Negative Prompt

## Director Rule

AIへ「演技」ではなく「感情」を伝える。

## Naming Rule

```text
CUT06_Dreamina.md
CUT17_Dreamina.md
```

## Tone Rule

FLYSTAR77作品は派手さより余韻を優先する。

大きな感情ではなく、静かな感情を描く。
