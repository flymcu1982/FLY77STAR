## 📋 初日の注意点

### Gmail 運用での確認ポイント

**重要度判定の目安:**
- 🔴 URGENT: 24時間以内に対応が必要・納期が迫っているなど
- 🟠 HIGH: 1-3日以内に返信が必要・重要な打診
- 🟡 MEDIUM: 3-7日以内でよい・情報系
- 🟢 LOW: 1-2週間以内・参考情報

**分類別の例:**
- WORK: 制作依頼・編集依頼など
- COLLAB: アーティスト・クリエイターからの協力打診
- PLATFORM: YouTube・TikTok・Instagram 自動通知
- SYSTEM: GitHub・Netlify・Google Cloud 通知
- FAN: ファンからのメール・応援
- SPAM: 迷惑メール

### SNS DM 運用での確認ポイント

**プラットフォーム別の確認方法:**

**TikTok:**
- アプリ「メッセージ」セクション
- または Web 版の「Messages」タブ

**Instagram:**
- アプリ「DM」セクション
- または Web 版 Business Suite の「メッセージ」

**X:**
- アプリ「メッセージ」セクション
- または Web 版の「メッセージ」タブ

### Q2: DM がない場合は？
**A:** 「新着 DM 件数: 0 件」と記入してください。チェックシートに「確認済み・メッセージなし」と記載してください。

### Q3: メール・DM を送信したい場合は？
