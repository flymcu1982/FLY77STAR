```
→【SNS-DM-001】【送信】で実際に送信します
```

### YU 承認実行 — SNS DM 返信承認

**実行者:** YU  
**ファイル:** `Edit/SNS_DM_APPROVAL_FLOW.md`

**利用可能なコマンド:**

| コマンド | 機能 | 例 |
|--------|------|-----|
| **【送信】** | 下書きをそのまま SNS に送信 | 【SNS-DM-001】 【送信】 |
| **【修正】** | 内容を修正して再提示 | 【SNS-DM-002】 【修正】「料金」→「予算帯」 |
| **【確認中】** | 確認中のまま保留 | 【SNS-DM-003】 【確認中】 |
| **【削除】** | 下書きを破棄・送信しない | 【SNS-DM-004】 【削除】 |

**実行例:**
```
【SNS-DM-001】 【送信】
→ DM-001 を即座に TikTok に送信

【SNS-DM-002】 【修正】「予算」→「ご予算帯」に統一
→ 修正を確認してから再度提示

【SNS-DM-003】 【確認中】
→ 確認中のままキープ。後日判定

【SNS-DM-004】 【削除】
→ 下書きを破棄。送信しない
```
