## 📅 実運用スケジュール

### 朝 09:00 — Daily Mail Review

**実行者:** YU  
**ファイル:** `Edit/DAILY_MAIL_REVIEW_CHECKLIST.md`

**実行内容:**
1. fly77star.official@gmail.com にログイン
2. 新着メール全件確認
3. 重要度・分類別に記入
4. ラベル振り分け実施
5. 返信対象メール抽出
6. 重要メール通知を作成

**出力:** チェックシートに数値・内容を記入

---

### 午後 15:00 — Response Draft Preparation

**実行者:** Claude Code (AI社員)  
**ファイル:** `Edit/AFTERNOON_DRAFT_PREPARATION.md`

**実行内容:**
1. 朝のチェックシートから返信対象メールを確認
2. 4 種類の返信テンプレートから選択
   - A: 仕事依頼への詳細確認返信
   - B: コラボレーション打診への返信
   - C: ファンメール返信
   - D: 対応不可の丁寧な断り
3. 各メール別に下書きを自動生成

**出力:** 返信下書き案（YU 確認待ち）

---

### 夜 18:00 — Daily Studio Report

**実行者:** Claude Code (AI社員)  
**ファイル:** `Edit/DAILY_STUDIO_REPORT_TEMPLATE.md`

**実行内容:**
1. 本日のチェックシートから数値を集計
2. 受信数・分類・ステータス別に集計
3. 重要メール（URGENT/HIGH）を記載
4. 返信下書き準備状況を記載
5. 特記事項を記載
6. 翌日への引き継ぎ事項を記載

**出力:** 日報を作成 → YU に報告
