**ステータス:** 設計フェーズ開始

## 📌 プロジェクト概要

### ビジョン

FLY77STAR スタジオの全メール・SNS DM を一つのダッシュボードで統一管理し、YU が全受信情報を効率的に一覧確認・優先度判定・返信管理できるシステムを実装する。

### 目的

1. **情報の一元化**: Gmail + TikTok + Instagram + X の DM を一箇所で確認
2. **優先度管理**: URGENT/HIGH/MEDIUM/LOW の重要度が一目瞭然
3. **返信効率化**: テンプレート・下書き・承認フロー統合
4. **分析・レポート**: 日報・週報・月報の自動生成
5. **パフォーマンス向上**: 確認時間 50% 削減目標

---

## 🎯 主要機能要件

### 1. 統合受信トレイ

**表示項目:**
- 受信日時（最新順）
- 送信者（メール/SNS ユーザー名）
- 件名 / メッセージ内容（プレビュー）
- プラットフォーム種別（Gmail / TikTok / Instagram / X）
- 重要度バッジ（🔴 URGENT / 🟠 HIGH / 🟡 MEDIUM / 🟢 LOW）
- ステータス（📥 INBOX / 👀 REVIEW / 📝 DRAFT / ✅ DONE）
- アクション buttons（返信・承認・スキップ・アーカイブ）

**フィルター機能:**
- プラットフォーム別フィルター
- 重要度別フィルター
- ステータス別フィルター
- 未読/既読フィルター
- 検索機能（件名・本文・送信者）

**表示モード:**
- リスト表示（デフォルト）
- カード表示（サムネイル対応）
- タイムライン表示（日付グループ化）

---

### 2. 自動分類・優先度判定

**分類ロジック（AI 社員が自動実行）:**
- メール件名・本文から重要度を自動判定
- プラットフォーム・送信者パターンから分類
- 既知の重要人物・重要キーワードを学習

**優先度判定ルール:**
- 🔴 URGENT（0-24h）
  - 返信要求のある依頼
  - 納期が明示されている案件
  - 重要人物からのメール
  
- 🟠 HIGH（1-3日）
  - メディア・協業打診
  - 重要な質問や問い合わせ
  - 作品・制作に関する打診
  
- 🟡 MEDIUM（3-7日）
  - ファンレター・感想
  - 情報提供・参考資料
  - 自動通知（YouTube・TikTok など）
  
- 🟢 LOW（1-2週間）
  - 参考情報・ニュースレター
  - SNS フォロー通知
  - システム通知

---

### 3. 返信下書き自動生成

**機能:**
- 受信内容から「最適な返信テンプレート」を自動選択
- テンプレートをカスタマイズした下書きを生成
- YU が確認・修正後に「【送信】」コマンドで発送

**テンプレート種別:**
1. 制作依頼 — 詳細確認・スケジュール照会
2. コラボレーション — 提案内容確認・相談
3. ファン対応 — 感謝・励まし
4. ビジネス提案 — 確認・検討予定
5. URGENT 対応 — 迅速な確認返信
6. 対応不可 — 丁寧な断り

---

### 4. YU 承認フロー

**コマンド体系:**
- 【送信】 — 下書きをそのまま送信
- 【修正】 — 内容を修正して再提示
- 【確認中】 — 保留・後日検討
- 【削除】 — 対応しない・破棄

**複数 DM 同時処理:**
```
【SNS-DM-001】 【送信】
【SNS-DM-002】 【修正】「予算」→「ご予算帯」に統一
【SNS-DM-003】 【確認中】
【SNS-DM-004】 【削除】
```

---

### 5. 日報・分析機能

**日報フォーマット（毎日 18:00）:**
- 受信件数（プラットフォーム別）
- 重要度別集計（URGENT / HIGH / MEDIUM / LOW）
- 分類別集計（WORK / COLLAB / FAN / PLATFORM / SYSTEM など）
- 対応状況（送信済み / 下書き待ち / 確認中 / 削除）
- 特記事項（クレーム / 重要提案など）
- 翌日への引き継ぎ

**週報・月報:**
- トレンド分析（曜日別・時間帯別の受信パターン）
- 返信率・応答時間の統計
- 重要度分布の推移
- 送信者分析（頻出人物・企業）
