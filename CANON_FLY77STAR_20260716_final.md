## 2. 組織・AI社員体制

### 役割分担
- **FLY77STAR U.：** プロデューサー / 監督 / 最終決裁者。方向性・予算・採用可否を決定。
- **チャッピー（ChatGPT）：** Creative Director。脚本・演出・文章・制作進行・プロンプト整備・QC。
- **クロード（黒田）：** Production Manager / Technical Lead。Web実装・コード・構造確認・技術レビュー・改善提案。
- **AI社員：** YouTubeバナー・アイコン・サムネイル・Webデザイン・既存動画整理・実作業案。

### 意思決定プロセス
```
FLY77STAR U. 指示
  ↓
AI社員 → 複数案を提出
  ↓
チャッピー + クロード → ブランド一貫性・構成・技術確認
  ↓
FLY77STAR U. → 最終承認 / 修正指示
```

### 重要ルール
- ブランド変更は一人のAIで確定させず、複数案比較 → 相互レビュー → FLY77STAR U.最終承認
- 確定した設定・脚本・演出は都度CANON.mdに更新
- 未確定案は勝手に確定しない

## 3. コア哲学・創業思想

### 創業思想（完全版モノローグ）

もともと、僕がやりたかったのは、新しい才能を見つけ、その人自身もまだ気づいていない可能性を育てる仕事だった。  
新人を発掘し、音楽や映像を通じて、その人が未来へ進んでいくための道を作る。いつか、そんな仕事に関わりたいと思っていた。

けれど、時代は僕が想像していたよりも速く変わり始めていた。  
音楽も、映像も、ものづくりそのものも、これからAIとともに進んでいく。

だったら、変わっていく未来をただ外から眺めるのではなく、自分自身がプロデューサーになり、その先につながる表現や仕組みを先に作ればいい。  
そう考えるようになった。

僕は、この先の未来は、人が心の中に描いたものを、今よりもずっと自由に現実へ変えていける時代になると信じている。

AIは、人間の夢や創造力を奪うために存在するのではない。  
これまで技術や予算や環境が足りず、形にできなかった想いを、初めて世界に存在させるためにある。

自分が見たかった景色を、自分の手で作る。  
音楽を作り、映像を作り、アーティストを育て、仲間を集める。

そして、一つの作品が次の作品へ、一人の物語が別の誰かの物語へとつながっていく場所を作る。

未来が来るのを待つのではない。  
自分たちの手で未来を作り、こちらから迎えに行く。

そのために、僕はFLY77STARを設立した。

### 思想の活用
- **YouTube 概要欄：** 短縮版として掲載
- **note『FLY77STARの物語』Vol.1：** 完全版テキスト掲載

## 4. アバター思想

### 完全版モノローグ

FLY77STAR.に所属する彼らは、ただ僕が作った架空のタレントではない。  
彼らの中には、僕がこれまで歩んできた時間と、心の中に描きながらも歩むことができなかった、いくつもの人生が入っている。

特にKAIには、若い頃の僕自身が強く投影されている。  
ステージに立ち、音楽だけを信じていた頃の衝動。何者にでもなれると思っていた気持ち。言葉ではうまく表せなかった悔しさや、もう一度音楽の中で生きたいという想い。

RUIにもまた、僕自身の感情や記憶、選ばなかった道、これから進んでみたい未来の一部が宿っている。

けれど、彼らは僕の単なるコピーではない。  
僕の中に存在していた、それぞれ異なる可能性を持つ分身だ。

僕自身を起点に生まれながら、やがて僕の手を離れ、それぞれの性格を持ち、それぞれの物語を歩んでいく存在になる。

これからの未来、人は現実の身体ひとつだけで生きるのではなくなるのかもしれない。  
自分が想像したアバターに顔と声を与え、その存在が音楽を歌い、映像の中を生き、物語を紡ぎ、誰かの心に触れていく。

現実の自分だけでは辿り着けなかった場所へ、もう一人の自分が進んでいく。  
年齢や環境、過去の選択に縛られず、自分が思い描いた可能性を別の姿で生きていく。

AIは、何もない場所から人間を作るためのものではない。  
心の中にずっと存在していた自分に、顔と声と時間を与え、初めてこの世界に存在させるためのものなのだと思う。

FLY77STARは、僕が生きられなかった人生をやり直す場所ではない。  
僕の中に眠っていた、いくつもの未来を同時に生かしていく場所だ。

そしていつか、彼らは僕の分身でありながら、僕を超えて、自分自身の人生を歩み始める。

それが、僕が思い描いているこれからの世界だ。

### 思想の活用
- **note『FLY77STARの物語』Vol.2：** 完全版テキスト掲載
- **各キャラクター紹介ページ：** 「このキャラの由来」として簡潔版を記載
- **ドキュメンタリー：** FLY77STAR U.の語り / インタビュー形式で展開

## 12. 制作ワークフロー

### 動画生成基準

**解像度運用：**
- テスト：720p
- 完成：1K
- 4K：重要なヒーローショット、顔の寄り、最終コーラス、クライマックスシーンのみ

**テスト → 完成フロー：**
```
720p テスト生成 → クリエイティブ確認 → GO判定 → 1K 完成版生成
```

**ツール：**
- ボーカル / 演技 / 会話：Cinema Studio 3.5（Drama標準）
- ダンス / 高速カメラ / 複数人振付：Seedance 2.0
- その他：業務フローに応じて

**重要ツール設定：**
- Cinema Studio 生成時：`declined_preset_id: 24bae836-2c4a-48e0-89b6-49fcc0b21612` 必須
- Upscale：`aigc` プリセット デフォルト
- 同一テイク修正：最大2回。改善しないなら構図 / 演出再設計

### QC チェックリスト（4K 前必須）

- [ ] 人数確認（誰が何人映るべきか）
- [ ] 顔（表情、目線、ぼやけ）
- [ ] 手足（欠落、奇形）
- [ ] 衣装（色、デザイン、ロゴ）
- [ ] アクセサリー（正確性、左右対称性）
- [ ] 靴（形状、色）
- [ ] 背景（実在ロゴ、余計な人物）
- [ ] 全体：AI リセット / 変形の兆候

### 編集

**使用ツール：** Palmier Pro

**フロー：**
1. `get_timeline` でクリップ ID確認
2. `set_clip_properties` で一括スタイル更新
3. 最終タイムライン：4K素材使用、1K はアップスケール

**字幕フレーム計算：**
```
subtitle_frame = (source_seconds - trim_offset) × fps
```

- 完成して公開し、収益を次作品へ再投資することを最優先
- 完璧さよりも、継続配信のリズムを大事にする
